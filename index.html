Make your own bot

This will guide you through making a private GitHub repository which is where the code for your bot will live. It will then show you how to copy some existing code into that repository in order to get you started.

Please use Google Chrome. Other browsers have not been tested.
  1. Click here to create new github repository. Give it a name like 'my_genghis_client' or 'genghis-bot-9000', and then make sure you click 'Private', so that nobody else can steal your code!
    Leave all the checkboxes unticked and click 'Create repository'.
  2. You'll be taken to a page with lots of stuff, and it probably has a big header that says 'Quick setup — if you’ve done this kind of thing before'. Ignore it all, copy the URL of that page and paste it below. (It should look something like 'https://github.com/MY-USERNAME-HERE/MY-REPOSITORY-NAME-HERE):
  3. Run the following lines in a *nix/MacOS terminal in the directory where you want your bot code to live. It will create a repository for you and copy some code to get you started, making sure to replace MY_USERNAME with your github username and MY_REPO_NAME with your repository's name: git clone --bare https://github.com/beyarkay/genghis_client.git
    cd genghis_client.git
    git push --mirror git@github.com:MY_USERNAME/MY_REPO_NAME.git
    cd ..
    rm -rf genghis_client.git
    git clone git@github.com:MY_USERNAME/MY_REPO_NAME.git
    cd MY_REPO_NAME
    git branch -M main
    git push -u origin main

Register your bot with the Genghis Server

In this step you add a deploy key to your private repository. This will let Genghis (and only Genghis) have read-only access to your repository so that it can fetch the code you write and execute it in an actual game. You should have already followed all the steps for 'Make your own bot'.
  1. First, go back to 'Make your own bot' and copy the URL of your github repository into the text field (I know you haven't done it yet).
  2. Now click: The three little dots in the top right > Settings > Deploy Keys > Add Deploy Key.
  3. Choose a title for the deploy key, something like 'Genghis Server Deploy Key', then click the button below to copy your generated public deploy key to your clipboard: Enter a valid GitHub URL under the 'Make your own bot' heading to generate your public deploy key Paste this public deploy key in the big text box labeled 'Key' on GitHub.
    This key will give the Genghis Battle System read-only access to your repository, so that it's code can be run and your bots can be included in battles.
  4. Click 'Add key', and then enter your password. You'll probably get an email letting you know that a new key has been added.
  5. Almost done! The Genghis Server knows about your repository, but is just checking that everything's in order and that it can read your repository correctly. This page will automatically update when you've been verified. Please make sure you can see the key you just created at your repository's Deploy keys tab under Settings.

    While you wait, you can continue to the next section if you'd like.
  6. All done! Your bot will be included in the next game which you can view by clicking the red link at the top of this page.
    Whenever a new game starts, Genghis will pull the changes from your private repository using the deploy key and then attempt to run the bot it finds in your repository.
    Simply pushing your changes to your github repository will cause them to be included in the next Genghis game.
    You're almost all setup, there's one last step that's required if you want to be able to test your bot before it gets included in a game.

Test your bot on localhost

So far you've got what you need to write your bot, and the Genghis Battle System will grab your code from the GitHub repository and run your bot against all the other bots, but you can't easily test your bot yourself, before it gets thrown into a game.

This step shows you how to download a copy the Genghis Server, so that you can start up your own game where you can set up your bot to fight against copies of itself, so that you can make sure your bot code does what you want it to do.
  1. Run the following lines in a linux/MacOS terminal to clone the genghis_server github repository. You can clone it from anywhere, but it'll be easiest if the genghis_server repository is in the same directory as your genghis client repository. git clone https://github.com/beyarkay/genghis_server.git
    cd genghis_server
    chmod +x build_game.py
     You should see some output like: Cloning into 'genghis_server'...

    Receiving objects: 100% (3988/3988), 1.88 MiB | 417.00 KiB/s, done.

  2. To start a game, run this command in the genghis_server directory: ./build_test_server.py server_state.json At the prompts, enter the location of the genghis_client repository you cloned earlier, and then the number of copies of your bot you want to create (or just leave the prompt blank to use the defaults specified in square brackets): At any time, enter a question mark (?) and then press for help.
